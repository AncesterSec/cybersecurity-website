CYBERSECURITY AWARENESS WEBSITE - SETUP INSTRUCTIONS

Project: CyberSafe - Cybersecurity Awareness & Education Platform

FILE STRUCTURE:
cybersecurity-website/
├── index.html          (Main HTML file)
├── styles/
│   └── style.css       (All CSS styles)
├── scripts/
│   └── main.js         (JavaScript functionality)
├── images/             (Directory for images)
└── README.txt          (This file)

SETUP INSTRUCTIONS:

1. Create a new folder on your computer for the website
2. Save each file in the appropriate location as shown above
3. For images, you can add:
   - favicon.ico (website icon)
   - hero-bg.jpg (hero section background - optional)
   - logo.png (website logo - optional)

4. Open index.html in a web browser to view the website

FEATURES INCLUDED:

- Responsive design that works on all devices
- Interactive cybersecurity quiz
- Smooth scrolling navigation
- Animated elements on scroll
- Mobile-friendly menu
- Professional cybersecurity color scheme
- Comprehensive educational content

CUSTOMIZATION:

You can easily customize:
- Colors by changing CSS variables in :root
- Content by editing the HTML
- Quiz questions in the quizData array in main.js
- Styling in style.css

DEPLOYMENT:

This is a static website that can be deployed on any web hosting service.
No server-side code required.

TECHNOLOGIES USED:
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome icons
